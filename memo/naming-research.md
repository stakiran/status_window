# なろう系キャラ名の調査

## 定番のネーミングパターン

### ナーロッパ風（西洋ファンタジー系）
- ゲルマン・北欧系: アルベルト、ジークフリート、エルザ、ブリュンヒルデ
- ラテン・イタリア系: レオナルド、クラウディア、ルシア
- 英語系: アレン、クレア、リリアン
- 混合造語: 実在の名前をベースに少し崩す（ライゼル、カイル、リーシェなど）

### 音の響き重視
- 2〜4音節が主流: 長すぎると読者が覚えられない
- 呼びやすい愛称を想定: アルベルト→アル、エリザベート→リーゼ
- 硬音（カ行・タ行）は男性的、柔音（ラ行・ナ行・マ行）は女性的に使われやすい

### 主人公の名前
- 転生者: 前世の日本名と異世界名の両方を持つパターンが多い
    - 異世界名だけで通すケースも多い（前世名は回想でのみ登場）
    - 前世名が平凡であるほど、転生前の「普通さ」が際立つ
- 短く呼びやすい: レン、カイ、ユウト、アレンなど

### ヒロイン名
- 可愛さ・美しさを音で表現: リア、フィーネ、セレナ、ミラ
- 花・宝石・天体由来: ローザ、ルビィ、ステラ
- お嬢様・貴族感: 長めのフルネーム＋愛称（リリアーナ→リリィ）

### 姓（ファミリーネーム）
- 貴族: 地名＋「フォン」「ヴァン」など（フォン・ヴァルトシュタイン）
- 平民: 職業由来（シュミット＝鍛冶、ミラー＝粉屋）や、姓なしも多い
- 冒険者: 姓より二つ名・通り名で呼ばれる

### 二つ名・通り名
- 「○○の△△」形式: 「紅蓮の魔女」「鉄壁の守護者」
- ギルドランクが上がると自然発生的に付く定番
- 本人が恥ずかしがるパターンはコメディ要素になる

### 避けるべきこと
- 読者が混乱する同音・似た名前の複数キャラ
- 長すぎて覚えられない名前（フルネームは長くても、普段呼ぶ名は短く）
- 日本語読みで語呂が悪い名前（黙読で引っかかると没入感が削がれる）

## 傾向まとめ

| 傾向 | 男性名 | 女性名 |
|------|--------|--------|
| 音節数 | 2〜4音節 | 2〜4音節 |
| 末尾の音 | ウ段・ス・ト等の硬め | ア・イ・ナ等の柔らかめ |
| 転生主人公 | 日本名そのままも多い | - |
| 異世界住人 | 欧風の響き | 欧風＋可愛い響き |
| 短さ | 2音が人気（レン、レム） | 2〜3音が人気（ユエ、ミリム） |

特にヒロインは2音節の短い名前が印象に残りやすく、多くの人気作で採用されている。

## Q&A

### Q. 名前の節が二つや三つある場合の背景と違いは？

西洋貴族名には構造があり、各パーツに役割がある。適当にくっつけるのではなく歴史的なルールに基づく。

基本形: 個人名 + 接続詞 + 家名

#### 接続詞（小辞）の意味

| 接続詞 | 言語圏 | 意味 |
|--------|--------|------|
| von（フォン） | ドイツ | 「～の出身」「～の領地の」 |
| van（ヴァン） | オランダ | 同上（ただし貴族とは限らない） |
| de（ド/デ） | フランス・スペイン | 同上 |
| di（ディ） | イタリア | 同上 |

「フォン・ヴァルトシュタイン」は「ヴァルトシュタイン領の（家の者）」という意味。

#### 名前が長くなるパターン

1. 領地の重複所有: 婚姻や相続で複数の領地を得た場合、両方の地名を名乗る
    - カール・フォン・アストラ＝ヴァルトシュタイン
    - →アストラ領とヴァルトシュタイン領の両方を持つ家
2. ミドルネーム: 洗礼名・聖人名を複数持つ
    - ヨハン・セバスティアン・フォン・ヴァルトシュタイン
    - →個人名が2つ（ヨハンとセバスティアン）
3. 母方の家名の併記: 名門同士の婚姻で、両家の名を残す
    - フォン・アストラ・ウント・ヴァルトシュタイン
    - →「アストラ家にしてヴァルトシュタイン家の」
4. 称号の挿入: 爵位や敬称が入る
    - グラーフ（伯爵）・フォン・ヴァルトシュタイン

#### なろう系での実用的な使い分け

| キャラの身分 | 名前の長さ | 例 |
|-------------|-----------|-----|
| 平民・冒険者 | 個人名のみ | カイル |
| 下級貴族・騎士 | 個人名 + von + 家名 | カイル・フォン・ベルク |
| 上級貴族 | 個人名 + ミドルネーム + von + 家名 | カイル・アレクシス・フォン・ベルク |
| 大貴族・王族 | さらに領地名や称号が加わる | カイル・アレクシス・フォン・ベルク＝ヴァイセン |

名前の長さ＝格の高さという暗黙の法則があり、読者も直感的に理解できる。
