# モンスター図鑑スキルの英語名候補

ポケモン図鑑的な「モンスター図鑑」をスキル名にしたい。ピクチャーブックは長いしダサい。

## 候補
- ベスティアリ（Bestiary） — 中世ヨーロッパの「動物・魔物誌」。ナーロッパとの相性抜群でモンスター図鑑の意味としてドンピシャ。短くて響きもいい。一番おすすめ
- コデックス（Codex） — 古代の写本・文献。「知識の集積」感がある
- コンペンディウム（Compendium） — 要覧・総覧。やや長いが厨二感がある
- アーカイブ（Archive） — 記録庫。馴染みがあって分かりやすい
- インデックス（Index） — 索引・目録。シンプルで短い
- レジストリ（Registry） — 登録簿。やや事務的だがゲーム的でもある
