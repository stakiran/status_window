# 痕跡の名前候補
モノマドで壁や地面をえぐって描く痕跡の名前。厨二寄りの命名センス（モノマド同様）を意識。

- 刻痕《グリフ》 - glyph（象形文字・刻印）。短くて響きがいい。「ダンジョン深層部にグリフを残す謎の存在」は語感として強い
- 刻印《スティグマ》 - stigma（烙印・聖痕）。厨二感が強く、宗教的な畏怖を感じさせる。発見者が「聖痕だ……」と畏れる展開に合う
- 墓碑銘《エピタフ》 - epitaph。倒した魔物への弔いとして刻む、という意味づけができる。ベスティアリとの相性が良い（図鑑に記録し、現地にエピタフを残す）
- 落書《グラフィティ》 - graffiti。バンクシーとの親和性が最も高い。ただし「落書き」だと軽すぎるかもしれない
- 刻紋《ルーン》 - rune（古代文字）。異世界ファンタジーとしては王道だが、既存作品で使われすぎてる感もある

## 所感
エピタフを推す。ベスティアリで魔物を記録→倒した現場にエピタフを刻む、という一連の流れが自然にできる。「未踏破エリアに墓碑銘が刻まれている」という発見は、冒険者にとって恐怖と好奇心の両方を煽れる。
