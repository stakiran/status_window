# なろう系とは

## 概要

「なろう系」とは、小説投稿サイト「小説家になろう」（https://syosetu.com/）を発祥とする、Web小説の一大ジャンル群の総称。同サイトで人気を博した作品群に共通するテンプレート的な要素・展開を持つ作品を指す。サイト外で発表された作品であっても、これらの特徴を備えていれば「なろう系」と呼ばれる。

## 代表的な特徴

- 異世界転生・異世界転移を起点とするストーリーが多い
- RPG的な世界観（レベル、スキル、ステータス、ダンジョンなど）を採用する
- 主人公が何らかのチート能力（規格外の強さ）を持つ
- 主人公視点の一人称で語られることが多い
- 章ごとに短いエピソードを積み重ねる連載形式
- 追放・ざまぁ・スローライフなど定番の展開パターンがある

## 主要なサブジャンル

- **異世界転生もの**: 現代日本で死亡し、異世界に生まれ変わる
- **異世界転移もの**: 何らかのきっかけで異世界に飛ばされる
- **悪役令嬢もの**: 乙女ゲームの悪役キャラに転生し、破滅フラグを回避する
- **追放もの**: パーティや組織から追放された主人公が、実は最強だったと判明する
- **スローライフもの**: 異世界でのんびり暮らすことを目指す
- **ざまぁもの**: 主人公を虐げた者たちが後に痛い目に遭う展開

## ステータスウィンドウとの関連

なろう系作品ではゲーム的な UI 要素がそのまま物語世界に存在するケースが多い。「ステータスウィンドウ」はその代表格で、キャラクターの能力値やスキルを表示する画面が、作中世界で実際に見える・操作できるものとして描かれる。本作「ステータスウィンドウ（物理）」は、この概念に「物理」という捻りを加えた作品である。

## 出版・メディア展開の流れ

1. 「小説家になろう」で連載（無料公開）
2. 人気が出ると書籍化（商業出版）のオファーが来る
3. コミカライズ（漫画化）される
4. アニメ化される

この流れで多数のヒット作が生まれており、現在のライトノベル市場において「なろう系」は主流の一角を占めている。
