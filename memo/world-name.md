# 世界名の検討

## 決定
**リーディア** に決定。「読む(read)」から。ステータスを読む文化が根付いた世界にふさわしい。

## 検討時の候補

### ラテン語・古典語ベース
- フェネストラ (Fenestra) — ラテン語で「窓」。ステータスウィンドウが世界の理である本作に直球
- スペクラ (Specula) — ラテン語で「鏡・観察」。ステータスを覗く行為のイメージ
- ファートゥム (Fatum) — ラテン語で「運命」。ステータスに定義される人生観

### 造語（響き重視）
- アルヴィスタ — 「視る(vista)」を軸にファンタジー感を足した造語
- グランディール — 壮大さを感じさせる響き
- エルディナス — 「エル」系はなろうで人気の響き

### コンセプト直結型
- パネリオン — 「パネル（窓）」＋ファンタジー語尾。ステータスウィンドウが当たり前にある世界らしさ
- リーディア — 「読む(read)」から。ステータスを読む文化 ← **採用**

## 命名の基準メモ
- カタカナ3〜5音が読みやすい（長すぎると読者が覚えない）
- 作中で頻出するなら短く、地の文で稀にしか出ないなら多少長くてもOK
- 隠し意味があると、後から種明かしできて楽しい
