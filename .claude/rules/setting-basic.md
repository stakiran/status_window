# 舞台設定

## ジャンル
- なろう系ライトノベル

## 人称
- 主人公の一人称視点
- 主語は「俺」
- ただし主人公以外の視点を書くときがあり、その場合は主語も変わる

## 方向性
- ナーロッパ（なろう系によくある異世界で中性ヨーロッパ風の世界観）
    - また RPG 的な要素（レベル、スキル、魔法、ステータス、ダンジョンなど）も実在する
- 緩めのチート。主人公は「ステータスウィンドウ（物理）」というチート能力を持っている。しかし万能でも世界最強でもなく、気を引き締めて生きねばならない
- スローライフ。主人公はひっそりと生きたい。興し（新たにチームや会社や国を立ち上げる）やダンジョン攻略は望んでおらず、一人暮らしと一人旅を好み、人との交流は店など形式的な塩梅で満足する
- 主人公への猜疑心。ヒロインが主人公の弱さに疑問を抱く状況は、読者が得るカタルシスの一つである
    - 例1: コイツ、本当は強いんじゃないの？
    - 例2: この人、ランクも低いし、信頼できるパーティーとパートナーすらいなくて惨めなのに、なんでこんなに平然としてるんだろ
- 緩めのハーレム。複数のヒロインから好意を寄せられる展開は必須であり、カタルシスの一つでもある。しかし恋人化、結婚、性行為の描写ほど強くはない。
- 「ステータスウィンドウ（物理）」の駆使。本作の差別化ポイントであり、このスキルを駆使した頭脳戦や心理戦、その他創意工夫はカタルシスの一つである
