# この世界について
名前:

- **リーディア**
- 言及するときは主人公目線では「異世界リーディア」であり、リーディアの人類目線では「リーディア」である

世界の仕様:

- 神々の世界
    - 神々が住まう世界（上位世界）がただ一つ存在する
    - 神はそれぞれ好き勝手に世界（下位世界）をつくって遊んでいる
        - たとえるなら、神は下位世界をつくって遊ぶプログラマー
    - 世界には魂を注入でき、魂一つが人間級の賢い生物一つに相当する。魂は上位世界の仕様であり、神々でもコントロールできないが、ある下位世界で果てた魂は上位世界に戻され、特定の神が入手でき、別の下位世界に放り込める（放り込まず貯めてもいい）。このとき、前世の記憶をリセットして放り込んだのが転生であり、リセットせずに放り込んだのが転移である
    - 転生と転移は記憶のリセットの違いだけであり、赤子として生まれるかどうかは関係がない。したがって赤子として生まれなおす場合、記憶がない場合（誕生転生）と記憶がある場合（誕生転移）がある。赤子ではなく前世の人体ごとそのまま持ってくる場合もある（新規転移）。既存の誰かを乗っ取る形で転移することもある（上書き転移）。既存の誰かが死んだときに魂を注入して蘇生する場合もある（上書き転生）。
    - リセットの可否は上位世界の仕様で決まり、解明されておらず、神でもコントロールできない。転移は非常にレアであり、何十万年以上生きた神でも一度もお目にかかれないことも多く、確率で言えば宝くじよりも低確率である。つまり大半は誕生転生か、上書き転生が起きる。
    - 転移はイレギュラーでもあり、その世界をつくった神自身も把握できないイレギュラー（バグ）が生じることが多く、ともすると世界のバランスを崩しかねない。しかし言語翻訳など、その世界で生きていく上のお膳立ては自動でインストールされる
- 前世（現代日本）
    - 神アマテルがつくった世界
- リーディア
    - 神リディオンがつくった世界

リーディアと主人公:

- 主人公は前世で死亡する（前世の背景は後述）
- 主人公の魂を入手したのが神リディオンだが、激レアな転移を引いてしまったため、バグも考えず喜んで即座に転移させてしまう
- 主人公は異世界リーディアに新規転移するが、モノマドというバグを得ている
